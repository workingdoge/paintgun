# Archived v1 Conformance Fixtures

This tree contains archived KCIR v1 fixture modes moved out of the active
conformance suite:

- `kcir-node`
- `core-verify`
- `nf-obj`
- `nf-mor`
- `opcode-obj`
- `opcode-mor`

`tests/conformance.rs` only scans `tests/conformance/fixtures/` for active
vectors. Keep archived vectors in this tree to avoid mixing v1 and v2
conformance scopes.
