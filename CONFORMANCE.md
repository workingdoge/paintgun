# Conformance Status (DTCG 2025.10)

This document tracks what TBP currently supports from the DTCG 2025.10 modules it relies on.

Status labels:
- `supported`: behavior is covered by conformance fixtures and CI
- `partial`: subset implemented; edge cases listed below
- `unsupported`: not implemented; deterministic errors expected

## Notes

- The conformance suite is table-driven under `tests/conformance/fixtures/*`.
- Golden updates can be performed intentionally with `TBP_UPDATE_GOLDENS=1 cargo test conformance_fixtures`.
- Unsupported behaviors should fail with deterministic diagnostics and remain listed in `known_unsupported.json` until implemented.
